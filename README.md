# colab
1. Colab не требует установки ничего на компьютер, работает прямо в браузере, даёт бесплатные вычислительные ресурсы и автоматически сохраняет в Google Drive.

2. Два типа ячеек: **код** — выполняется как Python, **текст** — оформление в формате Markdown для заметок и пояснений.

3. Функция создаётся через `def`, например:
```python
def say_hi(name):
    return f'Привет, {name}!'

print(say_hi("Александр"))
```

4. Использованные типы данных: `int` — целые числа, `float` — дробные, `str` — строки, `bool` — True/False.

5. Для ввода данных от пользователя используется функция `input()`:
```python
name = input("Введите имя: ")
age = int(input("Введите возраст: "))
```
